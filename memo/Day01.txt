- Java
	- 프로그래밍 "언어"
	- "컴파일" 언어
		- 컴파일이란 자바 언어를 컴퓨터가 알아들을 수 있게 한번 거치는과정
		
		- 인간이 보는 명령어를 컴퓨터가 알아들을 수 있는 언어(기계어)로 바꿔주는것. 이것을 compile이라고한다
	- 컴파일러 (compiler)
		- 컴파일을 해주게 하는 머신.
		- 통역가


	- 컴파일 언어 장점
		- 다양한 OS에 종속되지 않는 장점이 있다.
			- 다른 언어같은 경우, OS가 다른 컴퓨터에 실행하게 되면 재빌드를 해야하는 단점이 있다.

	- 컴파일 언어 단점
		- 시간이 다른 언어에 비해서 다소 느림
			- 최근 들어서는 속도 차이가 얼마 나지는 않는다


- 프로그래밍이란?
	- 어떤 문제나 목표를 달성하기 위해 나아가는 과정 ( it 언어를 사용하여 시스템이나 서비스를 만드는 과정)
	- 하나의 목표를 나아가기 위해 최적합되도록 만들어진 과정 

- Java 
	- 프로그래밍 언어
